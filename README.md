# Sales-Data-Analytic-Project
Sales Data Analysis using Python, Pandas and Matplotlib. Includes business insights and visualizations

## 📌 Project Objective
The objective of this project is to analyze sales data and generate business insights using Python.

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📂 Dataset Information
The dataset contains sales transaction data including:
- Order ID
- Order Date
- City
- Category
- Product
- Total Sales
- Profit

## 📈 Key Business Insights
- Total Revenue: ₹4,44,000
- Total Profit: ₹81,800
- Profit Margin: 18.42%
- Top Performing City: Bangalore
- Highest Revenue Category: Technology
- Most Profitable Product: Laptop

## 📊 Analysis Performed
- Monthly Sales Trend
- City-wise Sales Analysis
- Category-wise Revenue Analysis
- Product Profitability Analysis

## 🔎 Conclusion
Technology products contribute the highest revenue and profitability. Bangalore is the strongest performing market. Focusing on high-margin products can increase business growth.
